**ParserGenerator** <br>
Данный инструмент позволяет генерировать парсер языка, по описанию его грамматики. Для этого используется алгоритм LR(1)-анализа. На практике это означает, что достаточно составить формальное описание языка,
чтобы научиться распознавать программы на нем. <br> <br>
